The graph aboves shows the solution x(t) to the following second-order differential equation:

$$\frac{d^2x}{dt^2} = -\omega_0 x - \gamma \frac{dx}{dt} + F_0 \cos(\omega t)$$

You can adjust the parameters with the sliders above to observe the behavior of the solution when the parameters are changed. Initial conditions are

$$x(0) = 0; \frac{dx}{dt}(0) = 0$$

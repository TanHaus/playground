The Taylor series evaluated at a particular point is given by the following equation:

$$p(x) = f(x_0) + \frac{f'(x_0)}{n!}(x-x_0) + \frac{f''(x_0)}{n!}(x-x_0)^2 + \dots$$

In this web app, the derivative values are calculated using numerical methods.

TanHaus' Playground! (https://tanhaus.github.io/playground/)

# Description

Welcome to the TanHaus' Playground! This is where the Tans experiment and try out new ideas. It is a collection of web applets and small applications.

## Mathematical expressions

All mathematical expressions are written in LaTeX and rendered with [KaTeX](https://github.com/KaTeX/KaTeX). A small script in `/libraries/inlineKatex.js` renders inline mathematical expressions enclosed in dollar sign symbol `$`.
